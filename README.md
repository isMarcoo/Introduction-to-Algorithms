# Introduction-to-Algorithms: 《算法导论》学习笔记与代码

[![Language](https://img.shields.io/badge/Language-Python-blue.svg)](https://github.com/isMarcoo/CLRS-Solutions)

本项目用于记录阅读《算法导论》（Introduction to Algorithms, 4th Edition）的学习心得、课后习题（Exercises）以及思考题（Problems）的解答与代码实现。

---

## 📂 目录结构

项目按书本章节组织，每一章包含：
- `README.md`: 章节概述
- `xx-xx.md`: 核心概念，思考感悟。
- `xx-xx.py`: 对应小节例子代码实现（如有）。
- `practices/`: 练习题。
- `problems/`: 思考题。

```text

├── Part-4
│   ├── chapter-15
│   │   ├── README.md
│   │   ├── src/ (15-1.py 15-1.md)
│   │   ├── practices/
│   │   └── problems/
├── utils/
└── README.md
```
